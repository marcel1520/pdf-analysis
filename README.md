Please follow the following 7 steps to run this application locally:

1. Clone repository
2. Choose the python interpreter (Python 3.13) in local machine
3. Create virtual environment: conda create —-name <env_name>
4. Activate the environment 
5. Run pip install -r requirements.txt
6. Run pip install —-upgrade openai
7. Insert your open api key in openai_utils.py line 12
